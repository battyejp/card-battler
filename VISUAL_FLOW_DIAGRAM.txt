INTERACTION SEQUENCE:
--------------------

1. USER GRABS CARD (vertical drag > 30px)
   → Card duplicates to center of fan
   → Original card removed from position
   → Duplicate card gets:
     - Forward tilt (-15°)
     - Scale increase (110%)
     - Gold glow effect
   → Background darkens
   → Yellow table area appears

2. USER MOVES CARD
   → Card follows cursor
   → Continuously checks intersection
   → Table area color changes:
     - YELLOW when outside drop zone
     - GREEN when inside drop zone

3. USER RELEASES CARD
   Option A - Over valid area (GREEN):
     → Card is played
     → Overlay disappears
     → Game continues
   
   Option B - Outside area (YELLOW):
     → Card returns to original position
     → Tilt resets to original angle
     → Scale resets to 100%
     → Glow removed
     → Overlay disappears
     → Card reinserted into fan


COLOR REFERENCE:
--------------
Background Overlay:  rgba(0, 0, 0, 0.71)       [Dark semi-transparent]
Yellow Glow:         rgba(255, 193, 7, 0.39)   [Warning/Dragging]
Green Glow:          rgba(76, 175, 80, 0.39)   [Success/Valid]
